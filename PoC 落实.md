
### 阶段 0：环境配置与项目初始化
**主管指令：** 首先，我们需要一个干净的隔离环境，防止库冲突。你的设备是 AMD 处理器，重点在于确保 `PyTorch` 运行在 CPU 模式。

1.  **目录结构**：创建项目文件夹 `AddressIntelligence`，并在内部建立 `data`（放数据）、`models`（放缓存）、`src`（放代码）三个子目录。
2.  **环境安装**：
    *   创建一个 Python 3.9 的虚拟环境。
    *   使用 `pip` 安装：`modelscope` (阿里模型库), `torch` (CPU版), `cpca` (基础解析), `faiss-cpu` (向量检索), `pandas`, `rapidfuzz` (快速模糊匹配)。
3.  **验证指令**：写一个简单的脚本检测 `torch` 是否能识别你的 CPU 核心。

### 阶段 1：构建测试数据集 (Ground Truth)
**主管指令：** 没有对比就没有发言权。你需要手动生成或导入 1,000 条测试数据。

*   **任务描述**：创建一个 `raw_data.csv`，其中必须包含以下四类“脏数据”：
    1.  **残缺型**：只有小区名和门牌号（如：张江路66号）。
    2.  **错误型**：省市区逻辑冲突或有同音错别字（如：江苏省上海市、北惊市）。
    3.  **非标型**：包含大量干扰字符（如：***上海市&&浦东新区###）。
    4.  **旧称型**：使用已经更名或撤县设区的旧称。
*   **提示词**：使用 `pandas` 加载数据，并预留一列 `cleaned_address` 用于存放最终结果。

### 阶段 2：开发模块 A - 基础清洗引擎 (Rule-based)
**主管指令：** 先用最轻量的方法解决 80% 的简单问题。

*   **任务描述**：编写一个 Python 类 `BaseCleaner`。
*   **逻辑提示**：
    1.  调用 `cpca.transform` 对原始字符串进行拆解。
    2.  编写正则过滤函数：去掉特殊符号、统一全角半角字符、将“幢/楼/单元”等词汇标准化。
    3.  **关键逻辑**：如果 `cpca` 拆解出的省市区为空，不要报错，将其标记为 `is_fuzzy=True`，留给下一关。
---

### 阶段 3：开发模块 B - MGeo 语义解析引擎 (AI-NER)
**主管指令：** 这是处理那 20% 疑难杂症的“手术刀”。

*   **任务描述**：编写 `MGeoEngine` 类，加载达摩院预训练模型。
*   **提示词 (核心逻辑)**：
    1.  在 `modelscope` 中查找任务类型为 `Tasks.sentence_embedding` 的 `nlp_mgeo_backbone_chinese_base` 模型。
    2.  **性能优化**：由于你没有 GPU，必须在 pipeline 初始化时强制指定 `device='cpu'`。
    3.  **批量处理**：不要一条条喂数据，写一个 `batch_process` 函数，每 32 条记录作为一组进行向量化，利用你 5800H 的多线程能力。
    4.  **输出**：将地址字符串转化为 768 维的向量（Embedding）。

---

### 阶段 4：开发模块 C - 向量检索与标准池对齐 (Faiss)
**主管指令：** 语义向量化后，我们需要在“标准池”里找到最接近的正确答案。

*   **任务描述**：实现基于 `Faiss` 的相似度检索。
*   **逻辑提示**：
    1.  **准备标准池**：先用 `cpca` 跑一遍你的“标准库”，生成标准地址的向量并存储在 `Faiss.IndexFlatL2` 索引中。
    2.  **检索逻辑**：将模块 B 输出的模糊地址向量，在 Faiss 索引中进行 `search`，取 Top-1 结果。
    3.  **阈值控制**：计算 L2 距离。如果距离超过设定阈值，标记为“极度模糊，需人工审核”。

---

### 阶段 5：开发模块 D - 异常检测与最终聚合
**主管指令：** 最后一道防线，确保逻辑闭环。

*   **任务描述**：编写校验逻辑。
*   **逻辑提示**：
    1.  **层级校验**：对比 `cpca` 解析出的省市区和 MGeo 补全后的省市区。如果两者冲突（例如：cpca 说在上海，MGeo 说在北京），触发异常预警。
    2.  **结果导出**：将原始地址、拆分后的省/市/区/街道/门牌、相似度分值、处理状态（成功/待人工）导出为 Excel。
